# Интернет-магазин одежды

Это адаптивный сайт-каталог одежды, созданный с использованием HTML, CSS (Grid/Flexbox) и JavaScript.
Выполнили Антонова Анна и Мелехова София 345 группа
## Что сделано:
* Главная страница с категориями (`index.html`)
* Каталог товаров (`catalog.html`)
* Корзина (`cart.html`)
* Страницы авторизации и регистрации

## Как запустить:
Просто откройте файл `index.html` в любом браузере или используйте расширение Live Server в VS Code.
